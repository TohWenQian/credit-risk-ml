# Credit Risk Analyser

Web application that predicts company credit risk using an XGBoost machine learning model, with LIME explainability charts.

## Tech Stack
Java · Spring Boot · Spring Security · MySQL · Python · XGBoost · LIME · Thymeleaf · Bootstrap

## Features
- Company financial data input
- ML-powered credit risk prediction (6 categories: Lowest Risk → Default)
- LIME explanation chart showing which features influenced the prediction
- Role-based access control (Admin vs User)
- Member registration and authentication
